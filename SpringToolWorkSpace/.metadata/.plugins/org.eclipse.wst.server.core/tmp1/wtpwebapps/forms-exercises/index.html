Exercises: Form Handling

Note that for simplicity I am using HTML 4 and omitting style sheets. Later examples (and the entire Ajax tutorial) use xhtml (or HTML 5) and style sheets.
  • Input form for ThreeParams servlet (with GET)
  • Input form for ThreeParamsPost servlet (with POST)
  • Input form for ThreeParamsFiltered servlet (with filtering of special HTML characters from the user data)
  • Input form for Registration servlet (with with default values for data not supplied by user)
  • Input form for RegistrationFiltered servlet (with with default values for data not supplied by user plus filtering of special HTML characters)
  • Servlet that displays input form, plus redisplays forms that have missing data (with error messages and correct values re-inserted)
